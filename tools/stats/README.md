# Setup

1. Download JSONs from #game-reporting
2. Place the JSONs inside `jsons` folder
    - Create the folder if necessary
    - Delete any previous content if necessary
3. Add to `codes.txt` the teams
    - Create the file if necessary
    - Add subs if necessary
3. Run `ngm-stats.py`
    - If you simply want stats without sending to sheet, run `ngm-local.py`
4. Follow instructions on screen
5. No need to screenshot or go to the main sheet, you can just grab the generated screenshots inside your folder

# How to +1

1. Go to the JSON
2. Add the correct player to `correctGuessPlayers`
    - Follow the formatting
    - Order does not matter
3. You have finished
